# tldscan

## Installation
```
go install github.com/tomnomnom/hacks/filter-resolved@latest

git clone https://github.com/rix4uni/tldscan.git
cd tldscan && chmod +x tldscan
bash tldscan -h
```

## Usage
```
Quick-Mode Run:
   bash tldscan -q google

Verbose-Mode Run:
   bash tldscan -v google

Show Help:
   bash tldscan -h
```
